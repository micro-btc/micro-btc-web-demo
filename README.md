# micro-btc web demo

## About

This is a web demo of [micro-btc-signer](https://github.com/paulmillr/micro-btc-signer) by [Paul Miller](https://github.com/paulmillr).

## Usage

### Installation

First, clone this repo.

Second, install the dependencies:

```bash
npm install
```

### Running the app

```bash
npm start
```

### Testing

```bash
npm test
```

### Building for production

```bash
npm run build
```

### Ejecting

```bash
npm run eject
```
